## Usage Examples

### Basic Padding and Margin
```html
<div class="p-4 m-2">
  Element with 16px padding and 8px margin
</div>
```

### Responsive Spacing
```html
<div class="p-2 md:p-4 lg:p-8">
  Responsive padding: small on mobile, larger on desktop
</div>
```

### Centering with Auto Margins
```html
<div class="mx-auto max-w-md">
  Horizontally centered container
</div>
```

### Space Between Children
```html
<div class="flex space-x-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

### Negative Margins for Overlapping
```html
<div class="grid grid-cols-3 gap-4">
  <div class="bg-blue-500 p-4"></div>
  <div class="bg-red-500 p-4 -ml-2">Overlaps previous element</div>
  <div class="bg-green-500 p-4"></div>
</div>
```
