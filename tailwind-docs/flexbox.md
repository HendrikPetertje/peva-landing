# Flexbox

## Flex Direction

Control the direction of flex items.

| Class | CSS | Description |
|-------|-----|-------------|
| `flex-row` | `flex-direction: row;` | Place flex items horizontally |
| `flex-row-reverse` | `flex-direction: row-reverse;` | Place flex items horizontally in reverse |
| `flex-col` | `flex-direction: column;` | Place flex items vertically |
| `flex-col-reverse` | `flex-direction: column-reverse;` | Place flex items vertically in reverse |

## Flex Wrap

Control how flex items wrap.

| Class | CSS | Description |
|-------|-----|-------------|
| `flex-wrap` | `flex-wrap: wrap;` | Allow flex items to wrap |
| `flex-wrap-reverse` | `flex-wrap: wrap-reverse;` | Wrap flex items in reverse |
| `flex-nowrap` | `flex-wrap: nowrap;` | Don't allow flex items to wrap |

## Flex

Control how flex items grow and shrink.

| Class | CSS | Description |
|-------|-----|-------------|
| `flex-1` | `flex: 1 1 0%;` | Allow item to grow and shrink |
| `flex-auto` | `flex: 1 1 auto;` | Allow item to grow and shrink based on initial size |
| `flex-initial` | `flex: 0 1 auto;` | Allow item to shrink but not grow |
| `flex-none` | `flex: none;` | Prevent item from growing or shrinking |

## Flex Grow

Control how flex items grow.

| Class | CSS | Description |
|-------|-----|-------------|
| `grow` | `flex-grow: 1;` | Allow item to grow |
| `grow-0` | `flex-grow: 0;` | Prevent item from growing |

## Flex Shrink

Control how flex items shrink.

| Class | CSS | Description |
|-------|-----|-------------|
| `shrink` | `flex-shrink: 1;` | Allow item to shrink |
| `shrink-0` | `flex-shrink: 0;` | Prevent item from shrinking |

## Justify Content

Control how flex items are positioned along the main axis.

| Class | CSS | Description |
|-------|-----|-------------|
| `justify-normal` | `justify-content: normal;` | Pack items normally |
| `justify-start` | `justify-content: flex-start;` | Pack items toward start |
| `justify-end` | `justify-content: flex-end;` | Pack items toward end |
| `justify-center` | `justify-content: center;` | Pack items around center |
| `justify-between` | `justify-content: space-between;` | Distribute items evenly with space between |
| `justify-around` | `justify-content: space-around;` | Distribute items evenly with space around |
| `justify-evenly` | `justify-content: space-evenly;` | Distribute items evenly |
| `justify-stretch` | `justify-content: stretch;` | Stretch items to fill container |

## Justify Items

Control how grid items are aligned along their inline axis.

| Class | CSS | Description |
|-------|-----|-------------|
| `justify-items-start` | `justify-items: start;` | Justify items to start |
| `justify-items-end` | `justify-items: end;` | Justify items to end |
| `justify-items-center` | `justify-items: center;` | Justify items to center |
| `justify-items-stretch` | `justify-items: stretch;` | Stretch items |

| Class | CSS | Description |

| `justify-self-center` | `justify-self: center;` | Justify to center |

## Usage Examples

### Basic Flex Container
```html
<div class="flex">
  <div class="flex-1">Item 1</div>
  <div class="flex-1">Item 2</div>
  <div class="flex-1">Item 3</div>
</div>
```

### Centering Content
```html
<div class="flex items-center justify-center h-screen">
  <div>Perfectly centered content</div>
</div>
```

### Responsive Flex Direction
```html
<div class="flex flex-col md:flex-row">
  <div class="flex-1">Column on mobile, row on desktop</div>
  <div class="flex-1">Column on mobile, row on desktop</div>
</div>
```

### Flex Order
```html
<div class="flex">
  <div class="order-3">First in source, third visually</div>
  <div class="order-1">Second in source, first visually</div>
  <div class="order-2">Third in source, second visually</div>
</div>
```
