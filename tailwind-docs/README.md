# Tailwind CSS Documentation Structure

This folder contains comprehensive documentation for Tailwind CSS utility classes, organized by category. Each file contains all relevant utility classes for that category along with their CSS equivalents.

## Categories

### Layout
- **[Layout](./layout.md)** - Display, positioning, floats, clear, isolation, object-fit, object-position, overflow, overscroll-behavior, z-index

### Flexbox
- **[Flexbox](./flexbox.md)** - Flex direction, wrap, basis, grow, shrink, order, justify-content, align-items, align-content, align-self, place-content, place-items, place-self

### Spacing
- **[Spacing](./spacing.md)** - Padding, margin, space between

### Sizing
- **[Sizing](./sizing.md)** - Width, min-width, max-width, height, min-height, max-height, size

### Colors
- **[Colors](./colors.md)** - Text color, background color, border color, divide color, outline color, ring color, ring offset color, shadow color, accent color, caret color, fill, stroke
