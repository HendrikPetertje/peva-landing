## Usage Examples

### Responsive Sizing
```html
<div class="w-full md:w-1/2 lg:w-1/3">
  Full width on mobile, half on tablet, third on desktop
</div>
```

### Fixed Dimensions
```html
<div class="size-16">
  Square element: 64px Ã— 64px
</div>
```

### Constrained Containers
```html
<div class="max-w-4xl mx-auto">
  Centered container with max width
</div>
```

### Viewport-Based Heights
```html
<div class="min-h-screen">
  At least full viewport height
</div>
```

### Flexible Content
```html
<div class="w-max">
  Width adjusts to content
</div>
```

### Custom Arbitrary Values
```html
<div class="w-[32rem] h-[600px]">
  Custom dimensions using arbitrary values
</div>
```
