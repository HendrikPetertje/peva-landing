# Colors

Tailwind CSS includes a comprehensive default color palette with 22 colors, each with 10-11 shades. All colors are defined using the Oklch color space for consistent visual perception.

## Default Color Palette Scale

Each color comes with the following shades:
- `50` - Lightest shade
- `100` - Very light
- `200` - Light
- `300` - Light medium
- `400` - Medium light
- `500` - Base color (default)
- `600` - Medium dark
- `700` - Dark
- `800` - Very dark
- `900` - Darkest
- `950` - Extremely dark (not available for all colors)

## Text Color

Apply colors to text elements.

### Basic Colors
| Class | CSS | Description |
|-------|-----|-------------|
| `text-inherit` | `color: inherit;` | Inherit color from parent |
| `text-current` | `color: currentColor;` | Use current color |
| `text-transparent` | `color: transparent;` | Make text transparent |
| `text-black` | `color: var(--color-black);` | Black text |
| `text-white` | `color: var(--color-white);` | White text |

All color classes support the full range: `text-{color}-{shade}` where shade is 50-950.

## Usage Examples

### Text Colors
```html
<p class="text-gray-900 dark:text-gray-100">
  Dark text in light mode, light text in dark mode
</p>
```

### Background Colors with Opacity
```html
<div class="bg-blue-500/20">
  Semi-transparent blue background
</div>
```

### Responsive and State Variants
```html
<button class="bg-blue-500 hover:bg-blue-600 active:bg-blue-700 text-white">
  Button with hover and active states
</button>
```

### Border Colors
```html
<div class="border border-gray-300 border-l-4 border-l-blue-500">
  Card with blue left border accent
</div>
```

### Custom Colors with Arbitrary Values
```html
<div class="bg-[#1da1f2] text-[color:var(--my-color)]">
  Custom Twitter blue background
</div>
```
